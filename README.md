slp
===

slp: Discrete Prolate Spheroidal (Slepian) Sequence  Regression Smoothers (1.0-2)

Description
===========
Interface for creation of 'slp' class smoother objects for use in 
Generalized Additive Models (e.g., packages 'gam' and 'mgcv').
